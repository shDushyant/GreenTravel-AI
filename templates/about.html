🌿 About GreenTravel AI

Intelligent Carbon Footprint Prediction for Business Travel

📌 Project Overview

GreenTravel AI is a Machine Learning based web application developed to predict the likelihood of a business trip having a high carbon footprint.

Users simply enter their trip details, and the application instantly predicts the carbon risk while providing sustainability recommendations.


🚀 Features

  • ✅ AI-based Carbon Risk Prediction
  • ✅ Real-Time Prediction using Flask
  • ✅ Sustainability Recommendations
  • ✅ Interactive Dashboard
  • ✅ Professional Web Interface
  • ✅ Business Travel Analysis

🛠 Technology Stack

  • 🐍 Python
  • 🤖 Scikit-Learn
  • 📊 Pandas
  • ⚡ Flask
  • 🌐 HTML5
  • 🎨 CSS3
  • 💻 JavaScript

🎯 Project Workflow

  1. User enters trip details.
  2. Flask receives the request.
  3. The trained ML pipeline preprocesses the data.
  4. The model predicts the carbon risk.
  5. The result page displays the prediction probability.
  6. Recommendations are generated for greener travel.

🌱 Goal

The objective of GreenTravel AI is to encourage sustainable travel decisions by helping organizations identify trips that are likely to produce higher carbon emissions.
